# 配置系统支持的配置类型
- 配置系统的配置项是ConfigVar< T >，这是一个模板类，T的可能类型即为配置系统支持的类型
- T的可能类型有:
	1. 基本标量类型 int, float, long等
	2. STL容器类型
		1. std::vector
		2. std::list
		3. std::map
		4. std::unordered_map
		5. std::unordered_set
	3. 自定义类型SelfDefine
	4. STL类型和自定义类型的嵌套如std::vector< SlefDefine >
# 类型转换
对于上述T的可能类型，在YAML::node中
